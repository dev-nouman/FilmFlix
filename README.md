# Movie & TV Explorer

A sleek web app to browse movies and TV shows using a public API. Built with **plain HTML, CSS, and JavaScript**, featuring a responsive grid layout, pagination, and interactive search & filter options.

---

## **Features**

- **API Integration**: Fetches movies and TV shows from a public API.  
- **Grid Layout**: Clean, responsive grid to display content.  
- **Pagination**: Browse through multiple pages of results easily.  
- **Dark/Light Theme**: Toggle between dark and light modes.  
- **Search Bar**: Search movies/TV shows by name in real-time.  
- **Category Filter**: Filter content by genres like Horror, Romance, Comedy, etc.
